Intro to Reason

What is Reason?

-

Reason is OCaml

OCaml is Reason

How is Reason OCaml?

-
  • Reason is a new syntax
  • The syntax transforms to the same Abstract Syntax Tree (AST) as OCaml
  • It can compile to OCaml bytecode, native code or Javascript
What is OCaml?
-
Where did it come from?

Meta Language (ML)

  • Created by Robin Milner
  • Originally written in Lisp
  • Categorical Abstract Machine (CAM) evolved from work on ML
  • CAM and ML evolve into a new branch of languages called Caml
  • Objective Caml adds OO to Caml

What is OCaml?

  • Functional
  • Hindley Milner Type System (J. Roger Hindley & Robin Milber)
  • Structural Typing
  • Type Inference
  • Object Oriented (kinda)

Bucklescript

  • OCaml that can compile to JS
